Product App

A simple product management application built with **React**, **TypeScript**, **Vite**, and **Zustand**.  
It allows users to view product list and manage (view/edit) product details in a clean, component-based interface.

---

Tech Stack

| Category         | Technology                     |
| ---------------- | ------------------------------ |
| Frontend         | React 19 + TypeScript          |
| Build Tool       | Vite                           |
| Styling          | CSS                            |
| State Management | Zustand                        |
| Testing          | Vitest + React Testing Library |
| Linting/Format   | ESLint + Prettier              |

---

Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/kasiejro/product-app.git
cd product-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

App will be available at: http://localhost:5173

---

Running Tests

### Running unit tests

```bash
npm run test
```
